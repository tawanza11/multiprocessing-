# Multiprocessing Fibonacci

## คำอธิบาย
ตัวอย่างการใช้ `multiprocessing` เพื่อคำนวณเลข Fibonacci สำหรับหลายตัวเลขพร้อมกันโดยใช้ `ProcessPoolExecutor` ช่วยเพิ่มประสิทธิภาพเมื่อทำงานกับฟังก์ชันที่ใช้ CPU หนัก

## วิธีการใช้งาน
1. รันโปรแกรมด้วยคำสั่ง:
    ```sh
    python main.py
    ```

## คำอธิบายโค้ด
- ใช้ `ProcessPoolExecutor` เพื่อประมวลผลฟังก์ชัน `fibonacci()` แบบขนาน
- ใช้ `executor.map()` เพื่อคำนวณค่า Fibonacci สำหรับหลายตัวเลขพร้อมกัน
- แสดงผลลัพธ์ของแต่ละตัวเลขที่คำนวณได้ พร้อมเวลาในการประมวลผล

## ข้อดีของ Multiprocessing
- ทำให้การคำนวณที่ใช้ CPU หนักเสร็จเร็วขึ้น
- ใช้ประโยชน์จาก CPU หลายคอร์ได้เต็มที่
- เหมาะสำหรับงานที่ต้องใช้พลังประมวลผลสูง เช่น การคำนวณเชิงตัวเลข หรือ Machine Learning
